# FAI coursework — Group 6

Shared repo for **Fundamentals of AI** coursework. **Group 6** uses the letter grid image **`Images/Group_6.png`** only (other `Group_*.png` files from the course pack are not tracked here).

## Layout

| File | Author | Notes |
|------|--------|--------|
| `SectionA_Yoonjae.ipynb` | Yoonjae Lee | Slicing-based extraction; run with repo root as working directory |
| `SectionA_Jae.ipynb` | Weiqi Loo | Uses `skimage`; expects `Images/Group_6.png` |
| `SectionA_Mohamed.ipynb` | Mohamed Amin Mohamed Osman | Uses `skimage`; expects `Images/Group_6.png` |
| `Images/Group_6.png` | Course dataset | **Only** group image committed |

Open any notebook from the **repository root** so paths resolve to `Images/Group_6.png`.

## Dependencies

Python 3 with **numpy**, **matplotlib**, **Pillow** (for Yoonjae’s notebook), and **scikit-image** (for Weiqi and Mohamed’s notebooks):

```bash
pip install numpy matplotlib pillow scikit-image
```
